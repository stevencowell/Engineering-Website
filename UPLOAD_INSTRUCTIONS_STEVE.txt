GitHub Pages upload instructions

Upload the CONTENTS of this folder into the root of the GitHub repository.

Repository location:
stevencowell / Engineering-Website-main

What to do:
1. Open the GitHub repository.
2. Choose Add file / Upload files.
3. Drag all files and folders from inside this upload folder into GitHub.
4. Let GitHub replace existing files when names match.
5. Commit the upload.

Important:
- Upload the folder contents, not the folder itself.
- Keep the folder structure exactly as it is here.
- The folders with spaces in their names must stay named exactly:
  Year10 T1 Modules
  Year10 T2 Modules
  Youtube clips
